
# Oh No Games

Jeu développé avec **Unity** pour créer une expérience interactive et ludique.

## Description

Oh No Games est un jeu comportant plusieurs mini-jeux interactifs. Le projet a été réalisé dans le cadre d’exercices pratiques pour maîtriser Unity et le développement de jeux vidéo.

## Fonctionnalités

- Plusieurs mini-jeux intégrés  
- Système de score et suivi des performances  
- Interface utilisateur intuitive  
- Animations et interactions dynamiques  

## Stack technique

- **Moteur** : Unity  
- **Langage** : C#  
- **Plateformes cibles** : PC, Android, iOS (selon export)  

## Objectif

- Développer un jeu fonctionnel et attractif  
- Apprendre la gestion des scènes, des assets et des scripts dans Unity  
- Comprendre la logique de jeu et la gestion des scores
